# ⚠️ Experimental Code

This code is the first working version of Lisa's brain.

It runs. It remembers. It talks.
But it is early. Rough edges exist.

## What this means for you

- Run it at home, test it, break it
- Don't use it in production
- Don't expose it to the open internet without a firewall

## How you can help

- Test on your hardware and report what happens
- Fix bugs and open a pull request
- Improve the memory system
- Make the install smoother on different distros

## Known limitations

- Web interface is minimal
- Memory analysis can be slow on weak hardware
- No voice input yet
- No face/servo integration yet (coming separately)

The community builds this together.
If you find something broken — open an issue.
If you fix something — open a pull request.
